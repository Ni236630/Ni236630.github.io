GroceryList

For those who enjoy cooking but are too busy to plan dinners for a week, GroceryList allows a user to quickly create a grocery list from their uploaded recipes. My intention was to make an app that was highly personal and easy to use.

  • Styled using vanilla CSS
  • Used JavaScript for dynamic elements
  • Implementation of Fetch API to avoid page rereshing for data manipulation
